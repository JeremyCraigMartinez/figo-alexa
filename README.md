#How to run

```bash
# clone this repo
git clone git@github.com:JeremyCraigMartinez/figo-alexa.git
# run npm install
git clone https://github.com/matt-kruse/alexa-app-server.git
# run npm install
ln -s figo-alexa alexa-app-server/examples/apps
cd alexa-app-server/examples
node server
# and you're off!
```
